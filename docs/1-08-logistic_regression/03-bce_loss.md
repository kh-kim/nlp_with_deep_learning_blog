# BCE Loss

$$\begin{gathered}
0\le{P(\text{y}=\text{True}|x)}\le1 \\
\\
P(\text{y}=\text{True}|x)=1-P(\text{y}=\text{False}|x)
\end{gathered}$$

$$\begin{gathered}
\text{BCE}(y_{1:N},\hat{y}_{1:N})=-\frac{1}{N}\sum_{i=1}^N{
    y_i^\top\cdot\log{\hat{y}_i}+(1-y_i)^\top\cdot\log(1-\hat{y}_i)
}
\end{gathered}$$
